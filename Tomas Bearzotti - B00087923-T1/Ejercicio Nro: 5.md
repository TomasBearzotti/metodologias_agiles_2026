# Ejercicio Nro: 5

## Enunciado
Proponga cinco elementos que ayuden en el desarrollo de software referido a la propiedad de flexibilidad que se requiere.
## Resolución

Para lograr que un software sea flexible y se adapte rápido a los cambios, propongo estos cinco elementos:

- Modularidad: dividir el sistema en partes independientes para que cambiar un componente no afecte a los demás.
- Uso de patrones de diseño: aplicar soluciones ya probadas para que el código quede más estructurado y sea fácil de modificar.
- Pruebas automatizadas: tener tests armados permite hacer cambios en el código con la tranquilidad de que no rompimos nada viejo.
- Integración continua: automatizar la subida de código hace que incorporar nuevas funcionalidades o arreglos sea mucho más ágil.
- Refactorización: mejorar cada tanto la estructura interna del código sin cambiar su comportamiento, para que no se vuelva un desastre difícil de actualizar.
